# Digital Twin - Esteira de Produção

## 📌 Objetivo
Simular e analisar o comportamento de uma fila em uma esteira industrial.

## 📊 O que foi feito
- Simulação de chegada e processamento de itens
- Análise da fila (máxima, média, p95)
- Visualização do comportamento ao longo do tempo

## 📈 Principais insights
- Capacidade 10 → insuficiente
- Capacidade 11 → limite crítico
- Capacidade 12 → estabiliza, mas sob análise de picos

## 🛠️ Tecnologias
- Python
- Pandas
- NumPy
- Matplotlib
